# サウナ行った

サウナ後でもかんたんに記録できる、超シンプルなととのいログアプリです。

## 概要

「サウナ行った」は、サウナに行った日・施設名・ととのい度・セット数・サウナの種類を、できるだけ少ない操作で記録するためのアプリです。

サウナ後のぼんやりした状態でも使えるように、手動入力を最小限にし、選択式の入力を中心にします。

## コンセプト

サウナ記録をがんばらない。

細かい温度や時間を正確に残すことよりも、「今日サウナに行った」「どれくらい気持ちよかったか」を気軽に残せることを大切にします。

## このアプリで解決したいこと

既存の記録アプリは、入力項目が多くなると記録が面倒になりがちです。

サウナ後は気持ちよくぼんやりしているため、細かい入力は続きません。

このアプリでは、次のような体験を目指します。

- 手動入力をできるだけ減らす
- 3タップ程度で記録できる
- 書かなくても罪悪感がない
- 後から見返して「サウナ行ってるな」とわかる
- 記録内容をそのままXに投稿できる

## MVP機能

### サウナ記録

- 日付の自動入力
- 施設名の入力
- ととのい度の5段階選択
- セット数のプルダウン選択
- サウナの種類のプルダウン選択
- ひとことメモの入力

### 記録管理

- 記録一覧の表示
- 記録詳細の表示
- 記録の編集
- 記録の削除

### X投稿

- 記録内容からX投稿文を自動生成
- スマホの共有機能、またはX投稿画面を起動
- 投稿前にユーザーが文章を編集できる
- メモ内容は投稿文に含めない

## ととのい度

| 値 | 表示 |
|---|---|
| bad | 😵 しょんぼり |
| normal | 🙂 ふつう |
| good | 😌 まあまあ |
| totonotta | 🫠 ととのった |
| space | 🪐 宇宙 |

## セット数

| 値 | 表示 |
|---|---|
| 1 | 1セット |
| 2 | 2セット |
| 3 | 3セット |
| 4 | 4セット |
| 5_or_more | 5セット以上 |
| unknown | 覚えてない |

## 技術方針

初期開発では、ログイン機能やクラウド同期は実装しません。

まずはローカル保存で、サウナ記録の追加・表示・編集・削除ができる状態を目指します。

想定技術例:

- React Native
- Expo
- TypeScript
- AsyncStorage または SQLite

## 今後追加したい機能

MVP完成後に検討する機能です。

- 施設名の履歴候補表示
- 月ごとのサウナ回数表示
- よく行く施設ランキング
- お気に入り施設登録
- ダークモード対応
- バックアップ機能
- カレンダー表示
- サウナ飯メモ
- 入浴料金メモ

## 開発メモ

このアプリでは、高機能化よりも「記録が続くこと」を優先します。

機能を追加する場合は、次のルールを守ります。

- 入力項目を増やしすぎない
- 手動入力を増やしすぎない
- サウナ後でも使える画面にする
- 書かなくても成立する設計にする
- 一覧画面を情報過多にしない
