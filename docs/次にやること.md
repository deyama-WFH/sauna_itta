# 次にやること

## 1. リポジトリを作る

GitHubで新しいリポジトリを作成する。

おすすめリポジトリ名:

```text
sauna-itta
```

## 2. ローカルにクローンする

```bash
git clone <リポジトリURL>
cd sauna-itta
```

## 3. Expoプロジェクトを作る

まだ中身が空の場合は、以下のどちらかで作成する。

```bash
npx create-expo-app@latest .
```

またはTypeScriptテンプレートを明示する場合:

```bash
npx create-expo-app@latest . --template
```

## 4. mdファイルを配置する

以下のように配置する。

```text
README.md
docs/MVP仕様書.md
docs/Codex実装指示書.md
docs/実装タスク一覧.md
docs/Codex初回プロンプト.md
docs/次にやること.md
```

## 5. Codexを起動する

リポジトリのルートでCodexを起動する。

```bash
codex
```

## 6. Codexに初回プロンプトを貼る

`docs/Codex初回プロンプト.md` の中身をCodexに貼り付けて実装を開始する。

## 7. 最初の目標

最初の目標は、以下ができる状態。

- アプリが起動する
- ホーム画面が表示される
- 記録追加画面へ移動できる
- 記録を1件保存できる
- ホーム画面に戻ると記録が表示される

## 8. 最初のコミット例

```bash
git add .
git commit -m "docs: add MVP specs for sauna log app"
```

Expo初期設定後:

```bash
git add .
git commit -m "chore: initialize Expo app"
```

基本機能が入った後:

```bash
git add .
git commit -m "feat: add sauna log CRUD MVP"
```
