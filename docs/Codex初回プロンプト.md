# Codex初回プロンプト

以下をCodexに貼り付けて、実装を開始してください。

```text
あなたはReact Native / Expo / TypeScriptに詳しいエンジニアです。

このリポジトリで「サウナ行った」というサウナ記録アプリのMVPを実装してください。

まず以下のファイルを読んで、仕様を理解してください。

- README.md
- docs/MVP仕様書.md
- docs/Codex実装指示書.md
- docs/実装タスク一覧.md

## アプリの目的

サウナ後でもかんたんに記録できる、超シンプルなととのいログアプリを作ります。

## 最重要方針

- サウナ記録をがんばらない
- 手動入力をできるだけ減らす
- 選択式の入力を中心にする
- MVPに不要な機能は追加しない
- ログイン、クラウド同期、施設検索、X API連携は実装しない

## MVPで実装すること

- サウナ記録の追加
- 記録一覧
- 記録詳細
- 記録編集
- 記録削除
- ローカル保存
- X投稿文生成
- 共有機能

## 入力項目

- 日付
- 施設名
- ととのい度
- セット数
- サウナの種類
- ひとことメモ

## 実装順

1. プロジェクト構成を確認してください
2. 必要な依存関係を確認してください
3. 型定義を作成してください
4. 選択肢定数を作成してください
5. ローカル保存処理を作成してください
6. 画面を作成してください
7. X投稿文生成処理を作成してください
8. 共有機能を実装してください
9. 型チェックまたは起動確認をしてください

## 注意

一度に大きく作りすぎず、小さく実装してください。
仕様と矛盾する機能は追加しないでください。
実装後、変更したファイルと動作確認方法を説明してください。
```
